# SQL for Data Analysis – Ecommerce Dataset

## 📌 Project Overview

This project was completed as part of my **Data Analyst Internship Task 3**. The objective was to use SQL to extract, manipulate, and analyze data from an Ecommerce database.

The project demonstrates SQL concepts such as filtering, sorting, grouping, joins, subqueries, aggregate functions, views, and indexes.

---

## 📂 Dataset

The project uses a custom Ecommerce dataset consisting of three CSV files:

- customers.csv
- products.csv
- orders.csv

---

## 📌 Business Questions Answered

- Display all customer records.
- Find customers from a specific city.
- Identify high-priced products.
- Calculate total sales.
- Calculate average order value.
- Count total orders by payment method.
- Display customer order details using joins.
- Find products priced above the average price.
- Create a sales summary view.
- Improve query performance using indexes.
